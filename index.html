The Esports toolbox

The Esports Toolbox Limited is a space for Esports Organisations to meet their dream sponsors and for those same dream sponsors to Advertise!

A long term project where I developed the entire website from scratch. The website is built using the Django framework which allowed me to create a fully customised and bespokse website with unique features that I developed from scratch. Said website includes a Content Management system, Admin Dashboard, account permissions, Markdown rendering engine and an effiecient relational database on the backend.

Euphoria choir

A community based choir.

A choir paid me to build and design a website to put the choir on the map. The website achieved this by showcasing videos, images and audio of past performances. With an Events section showcasing where they would be performing next and the functionality to allow Choir members to login and be redirected to a private forum.

T3H Esports

Wordpress based eSports website.

Commissioned to create a WordPress website for an eSports organisation under the name of T3H eSports. The website showcased all of the organisations sponsors, teams, players, news and past matches that their teams had played. Utilised various WordPress ad-ons allowing for the admins to create and publish content without ever having to write any code.

Organisation shut down in 6th of October 2019 but you can take a view at the website via the Wayback Machine here. Bear in mind that aspects won't load nor work as it's just a snapshot taken and not the actual website.
